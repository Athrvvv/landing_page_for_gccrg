Let's Get Started!

Step 1: Check if Python is Installed

Download Python

If you don’t have Python installed, download it from the official Python installation page.

Step 2: Install the Software

Download Software

Download the software package from GitHub Releases and extract it to a folder.

Step 3: Unzip the Files

Extract the files and make sure both app.exe and prototype1.py are in the same folder.

Step 4: Launch the Application

Click on app.exe to run the application.

Setup Steps!

Step 1: Login or Signup

Login or create an account to begin using the application.

Step 2: Save Your Gesture Mappings

Save your custom gesture mappings or use the default ones.

Step 3: Run the Prototype

Click the Run Prototype button to start using the gesture controls.

Step 4: Open Any Game

Launch any game and use gesture controls to play.

Step 5: Stop the Prototype

Click the Stop Prototype button to end the session.
